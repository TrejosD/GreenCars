# Sistema de Mensajería

Proyecto estático desarrollado como práctica de maquetación con **HTML** y **CSS**, enfocado en el uso de **Flexbox** **Grid** y **Media Queries** para construir una pagina de una renta de autos.

## Descripción

La página muestra renta de autos con cuatro áreas principales:

- Un header de navegacion con el branding de la empresa.
- Una sección para ordenar los elementos del contenido principal.
- Un área principal donde mostramos los vehiculos disponibles para renta.
- El pie de pagina con las redes sociales de la empresa

El diseño utiliza imágenes locales, distribución flexible, layout responsivos al tamaño de la pagina botones, estados `hover` y `focus`, además de estilos personalizados para simular iconos de mensaje y menú.

## Tecnologías Utilizadas

- HTML5
- CSS3
- Flexbox
- Grid
- Media Queries
